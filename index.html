Real DevOps moment:

Code likha → kaam nahi kiya Fix kiya → aur kharab ho gaya Rollback kiya → kuch aur toot gaya

Aur end me: "Sir issue samajh aa gaya..." (actually nahi aaya hota)

Final truth by Sankalp Sir:

"DevOps me do hi log khush hote hain — Ek jo samajh gaya... aur ek jo samajh nahi paya"

CI/CD ki duniya bhi ajeeb kahani hai,
Build kabhi green toh kabhi pareshani hai...

Code likhte waqt lagta sab sahi chal raha,
Deploy hote hi pata chalta — system hi deewani hai

Pipeline pe bharosa kiya humne har baar,
Par error ne bola — "bhai main toh yahin ka permanent mehmaan hoon yaar!"
